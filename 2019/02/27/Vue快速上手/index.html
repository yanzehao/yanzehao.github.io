    安装

    Node.js

    • 这玩意用来做服务器开发的,不懂不要紧,其中的npm工具肯定是必不可少的

    • 既然是处于好奇心的目的,还是选择最新版吧,具体版本按需选择即可

    Vue Devtools

    • chrome插件,既然是官方文档推荐的,那不用多想,装就完事了

    Vue CLI 3.4.1 (2019-02-20)

    CLI 工具假定用户对 Node.js 和相关构建工具有一定程度的了解。如果你是新手，我们强烈建议先在不用构建工具的情况下通读指南，在熟悉 Vue 本身之后再使用 CLI。

    ——– 来自官方文档的善意提醒

    • 装上吧,人有多大胆,地有多大产!毕竟要恰饭的嘛~~~~

    项目初始化

    1. 创建一个实例

    2. 配置选择

    • 创建过程中有默认配置和自定义配置可供选择
    • 在自定义配置时发现好多不懂的模块,这并不要紧,按下A键全选就完事了
    • 创建并配置完成后可以保存配置,以便复用

    3. 启动项目

    • 需要下载vue/cli-service模块

    • (推荐)使用命令行启动本地服务器,默认端口8080 http://localhost:8080/

    • 还可以通过vue ui在本地启动UI界面,然后启动serve服务

    接下来就是逐渐熟悉的过程了,先从目录开始吧!
